PROJETOS

CadPy

Este projeto é um sistema de cadastro de usuários feito em Python com SQLite. Ele permite a criação, visualização, atualização e remoção de usuários com persistência de dados.

Calculadora

Este é um aplicativo simples de cálculo de IMC (Índice de Massa Corporal) criado com Python e PyQt5. Ele calcula o IMC e mostra uma seta indicativa com base na faixa de peso.

Piottin

Este projeto monitora o preço do Bitcoin utilizando a API CoinGecko e envia alertas por e-mail quando o preço cai abaixo de um limite definido pelo usuário.

QrPy

Este é um aplicativo em Python que gera um código QR a partir de um endereço web inserido no campo pelo usuário. Para interface gráfica foi utilizado o PyQt5.

PyGenius

Este é um jogo desenvolvido em Python com QtDesigner, inspirado no clássico jogo Genius dos anos 80. O objetivo é memorizar e repetir a sequência de cores.
